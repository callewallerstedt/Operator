# Operator Assist Web

Static site + serverless endpoint for human click assistance.

## Deploy (Vercel)
Set the project root to `assist_web/` and add:

- `DISCORD_WEBHOOK_URL` (required)

The Discord bot should set `ASSIST_WEB_URL` to the deployed URL.

## Local test
Use any static server and a local proxy for `/api/submit`, or deploy to Vercel.
